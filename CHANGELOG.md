# Changelog

This repo also publishes [GitHub Releases](https://github.com/epc314/oh-my-cassette/releases).
This file is the in-checkout release surface: `hermes plugins update cassette`
(or `git pull`) brings it along, so you can read the recent release arc
without leaving your install. Entries are generated by release-please from
conventional commits.

## 0.1.0 (2026-07-16)

Baseline release.

- Cassette video-editing plugin for Hermes Agent: media ingestion, asset
  manifests, edit-prompt preparation, and browser-driven Cassette jobs with
  progress notifications back to the chat gateway (QQ, Telegram, WeChat, web).
- Smart BGM matching via Jamendo and exact-BGM sources, plus the
  `cassette-video-edit` skill and `/cassette` commands.
- Guided installer (`scripts/install_plugin.py`), install diagnostics
  (`scripts/diagnose_install.py`), and a hosted web demo under `web_demo/`.
